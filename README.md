# Setup

You need to install globally @adonisjs/cli:

```bash       
npm i -g @adonisjs/cli                                        
```

Execute these commands one by one in back-end folder:

```bash
Copy .env.example to .env,                    
npm i                                         
adonis migration:run
adonis seed                               
```

# Start local server
Just run ```adonis serve ```
